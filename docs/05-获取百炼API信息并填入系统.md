# 05｜获取百炼 API 信息并填入系统

这是你接入我们的小程序系统的关键一步。

---

# 一、获取 API Key

1. 打开百炼平台  
2. 进入右上角头像 → 选择【API Key 管理】  
3. 复制 Key

（截图：API Key 页面）

---

# 二、获取你的 AI 专家的 ID

1. 左侧点击【智能体】  
2. 点击你创建的 AI 专家  
3. 进入“基本信息”页面  
4. 复制其中的 **Agent ID**

（截图：Agent ID 位置）

---

# 三、将 Key 和 Agent ID 填入我们的系统

在我们提供的后台管理界面中，找到：

【AI 设置 → 阿里云百炼配置】

填写：

- API Key  
- Agent ID  

点击【保存】即可。

---

现在，你的小程序已经可以使用你的 AI 回答了！
